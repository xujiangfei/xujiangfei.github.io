免费的腾讯云使用体验

发表于 2016-07-20   |   分类于 云服务器   |     |   阅读次数 
这两天申请了腾讯的云服务器体检，为期3天。服务器操作系统为CentOS 7.2 64位，默认账户为Root。

　　刚申请好云服务器体验之后，腾讯云项目组会给你发一份邮件，内容大致如下： 　　 　　 　

您新购买的云服务器(共1台)已分配成功（订单号：20160718xxxx），感谢您对腾讯云的支持！
服务器操作系统为    CentOS 7.2 64位 ，默认账户为 root ，初始密码为 xxxx
服务器名称          云主机ID          所在网络ID      内网IP           公网IP
未命名              ins-hws14c1w      基础网络     10.104.194.74    123.207.5.22

温馨提醒：
1. 若网站用于web访问，请及时备案。如需帮助，请查看备案论坛，或咨询腾讯云官方在线客服，
    您也可以拨打备案咨询电话：4009-100-100。
2. 如果您购买了数据盘，建议在服务器创建后首次登陆时，手动进行磁盘分区格式化操作。
    具体请参考 分区格式化操作指引。
3. 如您在云服务器使用中遇到业务环境部署、数据迁移等问题，您可以从腾讯云认证的代运维服务商处获得帮助，
    如需联系，请点击这里 。
谢谢！
立即查看使用教程，玩转腾讯云!

腾讯云项目组
2016.07.18

　　申请完了之后，我将系统重装了一下，并且选择了java环境的机器。找了好久之后，找到了tomcat的安装目录。进入bin目录，运行./startup.sh启动tomcat。 输入外网访问地址123.207.5.22，可访问到自己的网站。云服务器tomcat安装好之后默认有一个index，访问的网站内容大致如下：

恭喜您，您的网站已经成功开通!

此页面为新站点的默认页面。
如果您看到了此页，说明您的站点已经生效。您现在就可以开始上传您的网页了。
在您创建网站之前请您注意如下几点：
将此页替换为您的网站首页
您可以通过FTP将您制作好的网站程序发布到服务器空间的 / 目录中。
我们的服务器首页文件名搜索顺序为: index.html、index.jsp

谢谢您选择并信任康展云计算。 
顺祝商祺！

　　确实挺不错的，只需要将自己的web程序发布到tomcat下就可以访问自己的程序了。本来我就感兴趣云服务器到底是什么，弄完之后到没什么兴趣了。也就没有再弄ftp上传文件，毕竟只有三天免费体验。唯一的感受是：云服务器的速度真快，不到1分钟，大概30秒的时间就能重装系统。开启tomcat的时候，真的是秒开。关闭tomcat的时候，也是秒关。访问的时候速度也是杠杠的。打算闲的时候，做一个感兴趣的web程序，租一台服务器玩玩。说实话，如果不用的备案的话，早就租一台玩了。没时间去备案啊。
